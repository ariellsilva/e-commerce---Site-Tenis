# 👟 E-commerce Tênis

Projeto front-end inspirado em grandes e-commerces como Nike e Adidas, desenvolvido com HTML, CSS e JavaScript.

O sistema possui:
- Catálogo de tênis
- Filtros dinâmicos
- Login e cadastro
- Perfil de usuário
- Formulário de contato
- Layout responsivo

---

# ✨ Funcionalidades

## 🛒 Catálogo de Produtos
- Exibição dinâmica de tênis
- Cards modernos
- Modal de detalhes
- Organização por marcas

## 🔎 Sistema de Filtros
- Filtro por preço
- Filtro por tamanho
- Pesquisa por nome

## 👤 Sistema de Usuário
- Cadastro
- Login
- Logout
- Perfil do usuário

## 📩 Formulário de Contato
Integração com:
- Formspree

## 📱 Responsividade
Projeto adaptado para:
- Celulares
- Tablets
- Desktop
